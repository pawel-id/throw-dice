# Throw a dice

Returns random number 1 thru 6

## Install

```bash
npm i throw-dice
```

## Use

### module

```javascript
const throwDice = require('throw-dice')
console.log(throwDice())
// -> number 1 .. 6
```

### call directly
```bash
npx throw-dice
# -> number 1 .. 6
```

### install globally
```bash
npm i -g throw-dice
```

and call it anytime
```bash
throw-dice
# -> number 1 .. 6
```

short command version is also available
```bash
td
# -> number 1 .. 6
```
